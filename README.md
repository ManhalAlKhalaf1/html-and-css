# html-and-css
Welcome to my personal portfolio repository! This project showcases my abilities in frontend web development, focusing on responsive design, clean UI, and interactive elements.

## Project Overview

This repository contains the source code for my custom-built portfolio website. It was designed to highlight my skills using a unique "Glassmorphism" aesthetic, providing a modern and clean user experience. 

### Features Demonstrated:
* **HTML5 & CSS3:** Structural semantics and custom, from-scratch styling.
* **Fixed Navigation:** A sticky navbar with a frosted glass hover effect.
* **CSS Filters & Transitions:** Image manipulation (sepia/hue-rotation) and smooth hover states for text and overlay containers.
* **Media Integration:** Background videos that autoplay on loop (muted) with text overlays.
* **Layout Techniques:** Utilization of Flexbox for two-column layouts and structured data tables.
